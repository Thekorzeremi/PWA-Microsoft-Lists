# PWA-Microsoft-Lists
A PWA Version of Microsoft Lists

## How can I install it?
1. Go to the [website](https://pwa-microsoft-lists.vercel.app/)
2. Click on dot menu (...) on the top right corner of the browser 
3. Click on "Install PWA-Microsoft-Lists" or "Add to home screen"
4. Add it to your home screen 
5. Enjoy!
